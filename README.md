# Portfolio GitHub Pages — Alexandre Gauthier-Montan

## Mise en ligne

1. Créer un dépôt GitHub nommé `portfolio` ou `gustaviche.github.io`.
2. Ajouter les fichiers `index.html` et `style.css` à la racine du dépôt.
3. Aller dans **Settings > Pages**.
4. Dans **Build and deployment**, choisir :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
5. Sauvegarder.

Le site sera disponible après quelques minutes.

## Personnalisation rapide

- Remplacer les liens GitHub si les repos JobHunter AI ou Vault IA ont des URLs dédiées.
- Ajouter un fichier PDF du CV et un bouton vers celui-ci si besoin.
- Mettre à jour les métriques projets dès qu'elles sont stabilisées.
